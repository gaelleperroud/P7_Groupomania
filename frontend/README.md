# frontend

Here are the dependencies you'll need to install from within the frontend folder :

- axios
- luxon
- vue
- vue-router
- core-js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
